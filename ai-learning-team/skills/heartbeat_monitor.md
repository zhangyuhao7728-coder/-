# 心跳状态监视器 - 用例36

## 用途
监控所有自动化任务是否正常运行

## 触发词
- "心跳监控"
- "任务状态"
- "检查任务"

## 功能

### 1. 监控任务状态
- 读取 heartbeat-state.json
- 计算距上次运行时间
- 显示可读格式

### 2. 超时检测
- 默认阈值: 2小时
- 关键任务: 1小时
- 即将超时: 80%阈值

### 3. 自动警报
- 超时立即警报
- 可配置重新触发

## 显示格式
```
✅ 邮件检查: 4分钟前（正常）
⚠️ 日历检查: 28分钟前（即将超时）
🔴 新闻聚合: 3小时前（超时！）
```

## 命令

| 命令 | 说明 |
|------|------|
| `心跳监控` | 查看任务状态 |

## 文件
- 状态文件: `~/.openclaw/workspace/memory/heartbeat-state.json`
- 脚本: `scripts/heartbeat_monitor.py`

## Cron
- ⏰ 每30分钟检查一次
