# 知识图谱重建器 Skill - 用例40

## 用途
从日常记忆中提取实体和关系，构建知识网络

## 触发词
- "知识图谱"
- "查看知识网络"
- "查询知识"
- "更新图谱"
- "实体关系"

## 功能

### 1. 自动提取
- 从每日记忆提取实体（人物、项目、概念、工具）
- 自动识别关系（依赖、认识、使用、创建）
- 增量更新到知识图谱

### 2. 查询能力
- 查询项目依赖："哪些项目依赖 X 技术？"
- 查询人物关系："谁了解 Y 话题？"
- 查询概念关联："与 Z 概念相关的是什么？"

### 3. 统计展示
- 实体数量、关系数量
- 按类型分组显示
- 最近关系展示

## 命令

| 命令 | 说明 |
|------|------|
| `知识图谱` | 显示图谱概览 |
| `更新图谱` | 从今日记忆提取 |
| `查询 [关键词]` | 查询相关实体/关系 |
| `图谱统计` | 显示统计信息 |

## 文件位置
- 图谱数据: `~/.openclaw/workspace/memory/knowledge_graph.json`
- 脚本: `scripts/knowledge_graph.py`
