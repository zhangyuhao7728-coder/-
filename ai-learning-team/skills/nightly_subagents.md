# 7子Agent夜间并行作业 - 用例12

## 用途
每晚23:00启动7个并行子任务，次日早晨生成晨间汇报

## 触发词
- "夜间任务"
- "并行作业"
- "晨间汇报"

## 7个子任务

1. **记忆清理与整合** - 清理和整合今日记忆
2. **预算分析** - 分析项目预算使用情况
3. **技术调研** - 调研新技术/新模型
4. **书单推荐** - 推荐相关书籍
5. **AI优化研究** - 研究AI自身优化方法
6. **论文精读** - 精读AI相关论文摘要
7. **行为分析** - 分析顾问模式行为

## 工作流程

```
23:00 → 启动7个并行子任务
        ↓
    每个任务运行1小时
        ↓
    汇总结果
        ↓
次日 → 生成晨间汇报
```

## 命令

| 命令 | 说明 |
|------|------|
| `夜间任务` | 查看任务列表 |
| `夜间任务 run` | 手动运行 |
