## 📚 学习路径

### 阶段1: Python基础 (4周)
- curriculum/python-basics/
- 变量 → 循环 → 函数 → 类 → 异常 → 模块

### 阶段2: Python进阶 (4周)
- curriculum/python-advanced/
- 装饰器 → 生成器 → 异步 → 多进程 → 性能优化

### 阶段3: 算法与数据结构 (4周)
- curriculum/algorithms/
- 双指针 → 滑动窗口 → 二分查找 → 递归 → 动态规划

### 阶段4: AI/ML入门 (4周)
- curriculum/ai-ml/
- NumPy → Pandas → sklearn → PyTorch → Transformers

### 阶段5: 数据技能 (3周)
- curriculum/data-skills/
- 数据清洗 → 可视化 → 统计学 → 特征工程
