## 🎯 核心思想

```
算法 = 解决问题的方法 + 效率
```

---

## 📊 复杂度

| 复杂度 | 表示 | 速度 |
|--------|------|------|
| O(1) | 常数 | 最快 |
| O(log n) | 对数 | 快 |
| O(n) | 线性 | 一般 |
| O(n²) | 平方 | 慢 |
| O(2ⁿ) | 指数 | 最慢 |
