---

## 📖 变量

变量是存储数据的容器。

```python
# 变量赋值
name = "余豪"      # 字符串
age = 18            # 整数
height = 1.75      # 浮点数
is_student = True  # 布尔值
```

---

## 📋 基本数据类型

| 类型 | 示例 | 说明 |
|------|------|------|
| `int` | `18`, `100` | 整数 |
| `float` | `1.75`, `3.14` | 浮点数 |
| `str` | `"hello"` | 字符串 |
| `bool` | `True`, `False` | 布尔值 |
| `list` | `[1, 2, 3]` | 列表 |
| `dict` | `{"name": "Tom"}` | 字典 |

---

## 📝 变量命名规则

```python
# ✅ 正确
name = "Tom"
age18 = 18
my_name = "Tom"

# ❌ 错误
# 18age = 18      # 不能以数字开头
# my-name = "Tom" # 不能用连字符
# my name = "Tom" # 不能有空格
```

---

## 🔧 类型转换

```python
# 字符串转整数
int("18")  # 18

# 整数转字符串
str(18)    # "18"

# 字符串转浮点数
float("3.14")  # 3.14
```
