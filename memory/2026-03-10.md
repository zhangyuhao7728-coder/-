## AI Control Plane 项目

### 项目概述
- **位置**: ~/项目/Ai学习系统/ai-control-plane/
- **端口**: 3001
- **功能**: 统一管理本地+云端AI模型

### 架构优化
- 本地模型和云端模型完全分离管理
- 本地模型: 独立切换，互不影响
- 云端模型: 独立切换，互不影响
- 自动选择: 本地优先，本地不可用时自动切换云端

### 技术栈
- Node.js + Express
- Ollama 本地模型
- MiniMax 云端模型 (OpenClaw配置)

### 常用命令
```bash
cd ~/项目/Ai学习系统/ai-control-plane
npm start    # 启动服务
```

### API接口
- GET /api/models - 获取所有模型
- POST /api/local/switch - 切换本地模型
- POST /api/cloud/switch - 切换云端模型
- POST /api/chat - 对话测试
