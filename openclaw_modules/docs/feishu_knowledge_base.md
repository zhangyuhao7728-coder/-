# 飞书知识库 + OpenClaw 玩法

## 核心观点
- 未来无法与AI Agent交互的信息是"低效信息"
- 云端知识库 + 云端Agent 是未来趋势

## 飞书知识库搭建

### 1. 创建知识库
- 访问 https://ask.feishu.cn/
- 创建知识库，设置名称和描述

### 2. 付费设置
- 免费（公益）
- 兑换码（适合小红书等公域变现）
- 线上支付（私域直接付费）

## OpenClaw + 飞书玩法

### 1. 文档管理
- 创建飞书文档
- 设置权限为"编辑"或"管理"给OpenClaw
- 让OpenClaw读写文档

### 2. 应用场景
- 搬运官方Release Note
- 创建人物小传
- 搬运官方文档
- 定时扫描更新（每12小时）

### 3. Cron任务
- 每12小时扫描官方repo
- 有更新则搬运到知识库

## 坑和注意事项

1. 飞书文档写入顺序混乱
   - 官方feishu-doc skill不完善，需打补丁
   
2. 每个文档需要独立工作流
   - 需要大量调试
   
3. OpenClaw可能不按预期操作
   - 可能清空文档或填入错误内容
   
4. 稳定后要封装为skill

## 变现案例
- 有人付费加入知识库
- Dan Koe文章库（190篇）
