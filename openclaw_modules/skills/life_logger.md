# 生活记忆记录器 Skill - 用例60 (完整版)

## 用途
从对话中提取人物细节，构建「关系录」

## 触发词
- "人物库"
- "记住朋友"
- "添加人物"
- "查看人物"
- "关系提醒"
- "互动记录"

## 功能 (完整版)

### 1. 智能提取 ✅
- 过敏、喜好、厌恶
- 职业、学校、家乡
- 生日、电话、跟进

### 2. 关系管理 ✅
- 朋友、同事、家人
- 按关系分组显示

### 3. 互动记录 ✅
- 记录每次互动
- 追踪最后联系时间

### 4. 智能提醒 ✅
- 生日提醒
- 跟进事项
- 30天未联系提醒

### 5. 话题推荐 ✅
- 根据记忆推荐话题

### 6. 照片存储 ✅ NEW
- 添加人物照片
- 显示照片标识

### 7. 晚上自动提取 ✅ NEW
- 每天23:00自动运行
- 从对话提取人物信息

## 命令

| 命令 | 说明 |
|------|------|
| `人物库` | 查看所有人 |
| `添加人物 张三 过敏:花生` | 添加信息 |
| `查看人物 张三` | 查看详情 |
| `搜索 过敏` | 搜索关键词 |
| `关系提醒` | 查看待提醒 |
| `互动 张三 一起吃饭` | 记录互动 |
| `照片 张三 /path/photo.jpg` | 添加照片 |
| `自动提取` | 手动运行提取 |

## 数据存储
- 人物: `~/.openclaw/workspace/memory/people.json`
- 互动: `~/.openclaw/workspace/memory/interactions.json`
- 照片: `~/.openclaw/workspace/memory/photos/`

## 自动任务
- ⏰ 每晚 23:00 自动提取
