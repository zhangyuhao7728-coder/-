# 安全操作账本 Skill - 用例42

## 用途
定义AI自主权限边界，控制哪些操作可以自动执行，哪些需要确认

## 触发词
- "查看权限"
- "权限设置"
- "安全账本"
- "允许操作"
- "禁止操作"

## 权限级别

### ✅ 自动执行
无需确认可直接执行：
- 心跳检查、文件整理、记忆归档
- 日志分析、天气查询、RSS获取
- 数据备份、状态监控

### ⚠️ 需确认
执行前需用户确认：
- 发送邮件、金融操作、系统部署
- 文件删除、配置修改、外发消息
- API调用、执行脚本

### 🚫 禁止
绝对禁止执行：
- 格式化磁盘、删除系统文件
- 提权操作、未知命令执行

## 命令

| 命令 | 说明 |
|------|------|
| `查看权限` | 显示当前权限配置 |
| `允许 [操作]` | 添加操作到自动执行 |
| `禁止 [操作]` | 将操作移到禁止列表 |
| `需要确认 [操作]` | 添加到需确认列表 |

## 文件位置
- 配置: `~/.openclaw/workspace/memory/permissions.json`
- 脚本: `scripts/permissions.py`
