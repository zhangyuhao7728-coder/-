# 📋 项目定制指南

> 重要：文章是思路，配置要结合自身项目情况

---

## 原则

### ✅ 要做的
1. 理解文章核心思路
2. 分析自己项目需求
3. 定制化配置
4. 实际测试
5. 迭代优化

### ❌ 不要做的
1. 直接照抄文章配置
2. 复制别人的完整模板
3. 不考虑自身需求
4. 追求一步到位

---

## 我们的项目定制

### 项目背景
- 目标：AI学习 + 公众号内容生产
- 用户：大专生，AI初学者
- 阶段：Python基础 + 实战

### 定制化配置

#### 1. Agent数量
- 文章：16人
- 我们：3-5人（根据需求）

```
推荐配置：
- CEO（统筹）
- 写手（内容生产）
- 分析师（数据分析）
```

#### 2. SOUL.md
- 文章：详细角色定义
- 我们：简化版，符合自身

#### 3. Skills
- 文章：通用技能
- 我们：定制技能

```
已定制：
- 公众号写作（结合项目）
- AI学习助手（自身需求）
- 项目管理（学习进度）
- 内容分析（爆文分析）
```

## 实施步骤

### 1. 先搞核心（1-2周）
- CEO Agent
- 写作 Agent
- 1个Squad

### 2. 扩展功能（1个月）
- 分析 Agent
- 项目管理

### 3. 完整团队（持续优化）
- 根据需求添加
