**日期：** 2026年2月26日

## 文章简介

手把手教你用 Claude Code 搭配 WordPress Studio，靠提示词就能做出能用的插件，几分钟就能从想法落地，零基础也能上手！

---

## 完整操作步骤

### 1. 安装 Claude Code

1. 到 Claude Code 官网注册账号
2. 选择合适的付费套餐
3. 下载官方安装包安装
4. 一两分钟就装好

---

### 2. 安装 WordPress Studio

- 完全免费
- Mac 和 Windows 都能用
- 直接下载安装

**新建站点：**
- 名字随便取
- Studio 跑在本地电脑上
- 所有操作在自己电脑里
- 试错完全没风险

---

### 3. 启动 Claude Code，连到 Studio

1. 打开 WordPress Studio
2. 在"概览"标签找到「在…中打开」
3. 点「Terminal」（终端）
4. 输入命令 `claude`
5. 第一次用会让你登录 Claude 账号
6. 确认信任文件夹
7. 看到欢迎界面就准备好了

---

### 4. 开始第一个插件，纯靠说

**直接说你想要的插件功能：**

> "我现在在 WordPress 站点根目录，想要一个简单插件，能在网站后台顶部显示'你好，[我的名字]'的提示语"

**AI会自动：**
- 问几个小问题确认细节
- 建插件文件夹
- 生成遵循WP标准结构的插件文件

---

### 5. 激活插件，看看效果

1. 回到 WordPress Studio
2. 打开网站后台
3. 点「插件」
4. 找到刚生成的插件
5. 点激活

**效果：** 后台顶部显示 "你好，Nick"

---

### 6. 想加功能？继续跟 AI 说

- 加设置页面
- 加自定义区块
- 跟 AI 说清需求就行

---

## 技巧

| 技巧 | 说明 |
|------|------|
| 说清位置 | 跟 AI 说清在文件结构的哪个位置 |
| 描述具体 | 功能描述越具体越好 |
| 拆分需求 | 复杂功能拆成小需求，一步一步来 |
| 让AI解释 | 看不懂代码直接让 AI 解释 |

---

## 总结

**新手建议：**
1. 从简单插件开始试
2. 熟悉后再挑战复杂功能
3. 多试几次就知道怎么跟 AI 沟通

**环境配置好以后：**
- 想做插件直接跟 Claude 描述需求
- 会在 WordPress Studio 里生成可用代码
- 全程不用自己写多少代码
