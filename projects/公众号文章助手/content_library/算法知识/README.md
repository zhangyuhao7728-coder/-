## 数据结构

| 结构 | 操作 | 时间复杂度 |
|------|------|-----------|
| 数组 | 访问O(1)、插入O(n) | - |
| 链表 | 插入O(1)、查找O(n) | - |
| 栈 | push/pop | O(1) |
| 队列 | enqueue/dequeue | O(1) |
| 树 | 遍历 | O(n) |
| 图 | BFS/DFS | O(V+E) |
