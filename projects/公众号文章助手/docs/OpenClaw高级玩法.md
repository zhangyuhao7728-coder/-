## 一、入职流程

### 给AI一个真实身份
- 姓名
- 企业邮箱
- Google Workspace账户
- 加入公司通讯录和邮件组

### 邮件评分准则
- 客户合作匹配度
- 预算评估
- 合作意向
- 成交概率

### 自动工作流程
1. 收到邮件 → 全网搜索客户信息
2. 几分钟内生成评分简报
3. 80分以上客户 → Slack通知
4. 自动起草回复邮件
5. 绑定Gmail/HubSpot CRM/Telegram

## 二、防坑技巧（职场老油条）

### 1. 三层安全防护
| 层级 | 功能 |
|------|------|
| 第一层 | 代码清理，过滤脏指令 |
| 第二层 | 邮件关进沙箱隔离 |
| 第三层 | 最强AI模型扫描 |

### 2. 双重提示词栈
- 不同模型性格不同
- Claude喜欢自然表达
- GPT吃强硬指令
- 每个模型单独的员工手册

### 3. 提示词文件分离
```
agents.md - 执行流程
soul.md - 性格定义  
user.md - 老板偏好
```

### 4. 每晚同步审查机制
- AI自动校对所有指令文件
- 发现偏差Telegram提醒
- 说"修好它"自动修复

### 5. 消息过滤器
- 普通琐事：几小时汇总一次
- 急事：瞬间弹窗

### 6. 本地化资源优化
- Mac上用Nomic模型
- 零成本数据向量化

### 7. 自我修复功能
- event log.js记录调用轨迹
- 每天早晨AI审查错误日志
- learnings.md存储经验
