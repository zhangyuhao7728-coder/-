# WordPress Studio 配置指南

## 什么是WordPress Studio？

**WordPress Studio** 是 WordPress 官方（Automattic）推出的免费本地开发工具。

## 主要功能

| 功能 | 说明 |
|------|------|
| 一键创建站点 | 无需配置PHP/MySQL/Docker |
| 本地运行 | 所有操作在本地电脑 |
| 云端同步 | 可与WordPress.com同步 |
| 预览链接 | 生成云端预览 |
| AI集成 | 内置AI助手 |

## 首次配置

### 1. 创建第一个站点

1. 打开 WordPress Studio
2. 点击 "+" 新建站点
3. 输入站点名称（如：my-wp-site）
4. 等待创建完成

### 2. 打开开发环境

1. 站点创建完成
2. 点击站点
3. 点击「在…中打开」→「Terminal」
4. 终端会自动定位到站点目录

### 3. 连接Claude Code

在终端中输入：
```bash
claude
```

然后登录Claude账号即可开始开发！

## 常用操作

| 操作 | 说明 |
|------|------|
| 新建站点 | 点击 + 按钮 |
| 打开后台 | 点击站点 → 网站后台 |
| 打开前台 | 点击站点 → 访问站点 |
| 删除站点 | 右键 → 删除 |

## 与Claude Code配合

1. 打开Terminal
2. 输入 `claude`
3. 描述你想要的功能
4. AI会自动创建插件/主题

## 示例对话

```
你说: 帮我创建一个插件，在文章底部显示分享按钮

AI会:
- 创建插件文件夹
- 编写分享功能代码
- 告诉你在后台激活
```
