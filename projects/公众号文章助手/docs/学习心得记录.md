## 2026-03-14 | 两周内你能学会任何东西

**来源：** Dan Koe (微信公众号：逛逛GitHub)

### 核心观点

1. **技能 = 技巧叠加** - 不要整体学，要拆成小技巧
2. **从目标倒推** - 从"做成一件事"开始，不是从头学
3. **渐进超负荷** - 像健身一样，逐步加难度
4. **战术压力** - 把自己逼到绝境，效率最高

### 行动建议

| 阶段 | 行动 |
|------|------|
| 1 | 立即动手，不看教程 |
| 2 | 拆分目标为小技巧 |
| 3 | 设定2周完成一个作品 |
| 4 | 公开承诺，没退路 |

### 关键一句话

> 不要坐着慢慢学，要站起来快速做。
