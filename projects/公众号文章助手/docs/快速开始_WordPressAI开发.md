# 🚀 5分钟快速开始

## 你需要做的

### 步骤1：打开WordPress Studio
在 **Launchpad** 或 **Spotlight** 搜索 `WordPress Studio` 并打开

### 步骤2：创建站点
1. 点击界面上的 `+` 按钮
2. 输入站点名称（如：my-wp-site）
3. 等待创建完成

### 步骤3：启动开发
1. 点击刚创建的站点
2. 选择「在Terminal中打开」
3. 终端会打开并定位到站点目录
4. 输入 `claude` 并回车
5. 登录你的Claude账号
6. 开始描述你的需求！
