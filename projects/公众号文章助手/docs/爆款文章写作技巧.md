## 核心5条法则

### 1. 利他
- **不要自嗨**：写对读者有用的内容，不是自我表达
- **推己及人**：选题时问自己"多少读者会感兴趣？"
- **不藏私**：放心大胆分享，learn in public
- **积极回复**：回复读者提问，建立连接

### 2. 指向行动
- **福格行为公式**：B = M × A × P
  - 行动 = 意愿 × 能力 × 提示
- **关键**：读者不是没意愿，是没能力做
- **解决方案**：提供保姆级教程，手把手操作步骤
- **案例**：2万字部署教程成为爆款

### 3. 势大力沉
- **长文势能**：2万字、1万字系列
- **读者感受**：能感受到作者的用心
- **转发收藏**：即便读不完，也愿意转发
- **反面**：不要堆砌无用内容（屎山）

### 4. 勤奋
- **保持日更**：持续输出
- **海量阅读**：各平台保持信息摄入
- **Twitter技巧**：点赞收藏优质内容，算法会推送更多同类
- **提前预判**：比主流媒体早12-48小时发布

### 5. 工欲善其事必先利其器
- **AI辅助写作**：
  - 检索：深度调研
  - 头脑风暴：扩展思考
  - 编写：标题、行文润色
  - 审核：敏感内容、错别字
- **重要提醒**：
  - facts/记忆类 → AI可代劳
  - 认知深度 → 必须自己思考
  - AI想出来 vs 自己想出来 ≠ 一样

## 写作要点总结

| 要点 | 说明 |
|------|------|
| 利他 | 写对读者有用的 |
| 行动 | 提供可操作的步骤 |
| 深度 | 长文+保姆教程 |
| 勤奋 | 日更+信息摄入 |
| 工具 | AI辅助但不依赖 |
