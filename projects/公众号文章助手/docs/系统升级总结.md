## 二、可升级的功能

### 1. 提示词分离（重要！）

```
工作区文件结构：
├── SOUL.md      # AI性格/角色定义
├── USER.md      # 用户偏好设置
├── AGENTS.md    # Agent行为规则
└── memory/      # 记忆文件
    ├── 2026-03-14.md
    └── learnings.md
```

| 文件 | 作用 |
|------|------|
| SOUL.md | AI的角色定位、性格、说话风格 |
| USER.md | 用户的偏好、习惯、背景 |
| AGENTS.md | Agent的执行流程、任务分配 |
| memory/*.md | 按需读取的记忆，不要一次性塞入上下文 |

### 2. 定时审查机制

| 功能 | 实现 |
|------|------|
| 每晚同步审查 | Cron定时任务，检查指令一致性 |
| 自动修复 | 发现偏差自动修复 |
| 学习日志 | learnings.md 记录错误和修复 |

### 3. 消息过滤

| 消息类型 | 处理方式 |
|----------|----------|
| 普通消息 | 几小时汇总一次 |
| 急事/重要 | 立即弹窗通知 |

### 4. 自我修复

| 功能 | 说明 |
|------|------|
| 错误日志 | event log 记录每次调用 |
| 早晨审查 | AI自动审查昨晚错误 |
| 经验存储 | learnings.md 防止重复踩坑 |
