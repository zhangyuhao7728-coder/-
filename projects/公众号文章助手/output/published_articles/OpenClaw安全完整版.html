✅ 三、个人电脑防护方案（完整版）
1

保护API密钥

❌ 千万别这样：

# 危险！会泄露！ API_KEY = "sk-xxxxxx"

✅ 正确做法：

# 1. 创建 .env 文件 # 文件内容： API_KEY=sk-xxxxxx

操作步骤：

  1. 打开终端
  2. 输入：cd ~/.openclaw
  3. 输入：touch .env
  4. 编辑写入：API_KEY=你的密钥
2

只让自己使用

在 openclaw.json 中配置：

{ "telegram": { "allowFrom": [ "你的Telegram数字ID" ] } }

查ID：发消息给 @userinfobot

3

限制命令执行

创建白名单命令：

ALLOWED_COMMANDS = [ "python scripts/写作", "python scripts/分析", "python tools/安全检查.py" ]
4

文件访问隔离

只让访问项目目录：

import os PROJECT_ROOT = os.path.expanduser("~/你的AI项目") def validate_path(path): return path.startswith(PROJECT_ROOT)
5

启用安全沙盒

在配置中添加：

{ "sandbox": { "mode": "docker" } }
📊 四、检查配置效果

运行安全检查命令：

python3 ~/公众号项目/tools/安全检查.py

应该看到：

✅ Token管理: 已保护
✅ 用户白名单: 已配置
✅ 命令白名单: 已配置
✅ 文件隔离: 已配置
✅ 沙盒模式: 已开启
