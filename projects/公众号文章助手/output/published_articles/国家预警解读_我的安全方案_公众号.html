✅ 我的解决方案

## ✅ 我的解决方案

作为一个AI初学者，我是这样配置的：

1. 把密钥藏起来

### 1. 把密钥藏起来

❌ 千万别这样：

API_KEY = "sk-xxxxx"  # 写代码里＝自杀

✅ 正确做法：

# .env 文件
API_KEY=sk-xxxxx

2. 只允许自己用

### 2. 只允许自己用

ALLOWED_USERS = {8793442405}

def check_user(user_id):
if user_id not in ALLOWED_USERS:
raise PermissionError("Unauthorized")

3. 命令白名单

### 3. 命令白名单

Agent只能执行预设命令：

ALLOWED_COMMANDS = [
"python scripts/crawler",
"python scripts/analyze",
]

4. 文件系统隔离

### 4. 文件系统隔离

Agent只能访问项目目录：

PROJECT_ROOT = "~/你的AI项目"

def validate_path(path):
if not path.startswith(PROJECT_ROOT):
raise PermissionError("禁止访问！")

5. 启用安全沙盒

### 5. 启用安全沙盒

{
"sandbox": {
"mode": "docker"
}
}
