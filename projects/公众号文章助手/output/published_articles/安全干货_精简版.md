## ✅ 四、我的5个防护方案

### 4.1 方案一：保护密钥

**❌ 别这样做：**
```python
API_KEY = "sk-xxx"  # 危险！
```

**✅ 正确做法：**
```
# 创建 .env 文件
API_KEY=sk-xxx
```

---

### 4.2 方案二：只让自己用

```python
ALLOWED_USERS = {你的ID}

def check_user(user_id):
    if user_id not in ALLOWED_USERS:
        print("不允许！")
```

---

### 4.3 方案三：限制命令

```python
ALLOWED_COMMANDS = [
    "python scripts/写作",
    "python scripts/分析",
]
```

---

### 4.4 方案四：隔离文件

```python
PROJECT_ROOT = "~/我的项目"

def check_path(path):
    if not path.startswith(PROJECT_ROOT):
        return False
    return True
```

---

### 4.5 方案五：开沙盒

```json
{
  "sandbox": {
    "mode": "docker"
  }
}
```
