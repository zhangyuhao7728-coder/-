4️⃣ 5分钟搞定！保护你的钱

1

🔐 保护API密钥（最重要！）

❌ 千万别这样：

# 危险！别人能看到！ API_KEY = "sk-xxxxxx"

✅ 正确做法：

# 终端执行 touch ~/.openclaw/.env echo 'API_KEY=sk-你的密钥' >> ~/.openclaw/.env
2

🚪 只让自己进入

修改配置：

{ "telegram": { "allowFrom": ["你的ID"] } }

查ID：发消息给 @userinfobot

3

🛡️ 限制命令执行

创建白名单：

ALLOWED_COMMANDS = [ "python scripts/写作", "python tools/分析", ]
4

📁 文件访问隔离

只让访问指定目录：

PROJECT_ROOT = "~/公众号项目" def check_path(path): return path.startswith(PROJECT_ROOT)
5

📦 启用安全沙盒

加一行配置：

{ "sandbox": { "mode": "docker" } }

5️⃣ 一键检查

复制这条命令，粘贴到终端：

python3 ~/公众号项目/tools/安全检查.py

看到这些就是安全：

  • ✅ Token管理: 已保护
  • ✅ 用户白名单: 已配置
  • ✅ 命令白名单: 已配置
  • ✅ 文件隔离: 已配置
  • ✅ 沙盒模式: 已开启
