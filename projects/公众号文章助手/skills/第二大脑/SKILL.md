# 第二大脑 Skill

> 快速记录笔记和任务，像发短信一样简单

## 功能

- 快速记录想法
- 保存链接
- 创建任务提醒
- 搜索笔记

## 安全设计

- 📁 仅访问指定目录
- 🔒 不上传云端
- 📝 本地存储
- 🔍 可追溯

## 使用方法

```
记录：这个想法...
保存链接：https://...
提醒我：明天开会
```

## 安全配置

```json
{
  "allowed_directories": ["~/笔记/"],
  "max_file_size": "1MB",
  "blocked_extensions": [".exe", ".sh"]
}
```
