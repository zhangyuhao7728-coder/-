# 📝 教程类文章模板

> 用于：技术教程、功能介绍、使用指南

---

## 标题要求

- 简洁明了
- 体现价值
- 吸引点击
- 建议格式：xxx使用教程 / xxx完全指南 / xxx入门

示例：
- 《Python入门教程》
- 《OpenClaw安装配置完全指南》
- 《5分钟学会xxx》

---

## 开头模板

```
# 标题

> 一句话说明这篇文章解决什么问题

---

（介绍自己的身份/背景，引起共鸣）

大家好，我是xxx。

（说明为什么写这篇文章）

最近xxx，xxx，于是xxx。

今天把xxx分享出来，xxx也能学会。
```


## 写作技巧

| 技巧 | 说明 |
|------|------|
| 多用短句 | 一句话不要超过20字 |
| 多举例子 | 抽象概念具体化 |
| 多用emoji | 增加阅读趣味 |
| 代码要高亮 | 代码块要标注语言 |
| 重点突出 | 关键内容加粗或高亮 |

---

## 字数建议

| 类型 | 字数 |
|------|------|
| 短篇 | 800-1500 |
| 中篇 | 1500-3000 |
| 长篇 | 3000+ |

---

## 排版建议

- 段落不要过长
- 适当加空行
- 列表要用emoji
- 重要内容加引用块
