# 📝 经验类文章模板

> 用于：经验分享、踩坑总结、复盘反思

---

## 标题要求

- 体现价值
- 引起好奇
- 接地气

示例：
- 《我是如何xxx的》
- 《xxx踩坑合集》
- 《从零到xxx的进阶之路》

---

## 开头模板

```
# 标题

> 一句话概括核心经验

---

## 📌 背景

（说明分享的原因）

大家好，我是xxx。

最近xxx，xxx，于是想xxx。
```

---

## 内容结构

### 1. 问题/经历

```
## ⚠️ 遇到的問題/经历

（具体描述发生了什么）
```

### 2. 解决过程

```
## 🔧 我是怎么解决的/做了什么

### 尝试1：xxx

结果：xxx

### 尝试2：xxx

结果：xxx

### 最终方案：xxx

（详细说明）
```

### 3. 经验总结

```
## 💡 学到的经验/心得

1. xxx
2. xxx
3. xxx
```

---

## 结尾模板

```
## ✅ 总结

（简要回顾）

---

如果觉得有用，点个赞再走？
```

---

## 写作技巧

| 要素 | 说明 |
|------|------|
| 真实 | 不要编故事 |
| 具体 | 有具体细节 |
| 反思 | 有深度思考 |
| 接地气 | 用口语化表达 |

---

## 字数

800-2000字为宜
