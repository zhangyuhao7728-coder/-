# ⚙️ Systems - 系统层

> 整合多个子系统的一站式平台

## 系统架构

```
systems/
│
├── learning-system/      # 学习系统
├── coding-system/      # 编程系统
├── research-system/    # 研究系统
├── automation-system/  # 自动化系统
└── productivity-system/ # 效率系统
```

---

## 📚 learning-system

```
learning-system/
├── curriculum/    # 课程体系
├── problems/     # 题库
├── projects/     # 项目
├── dashboard/    # 仪表盘
└── ai-coach/    # AI教练
```

功能：
- Python/AI 学习
- 算法训练
- 进度追踪

---

## 💻 coding-system

```
coding-system/
├── editor/       # 代码编辑器
├── runner/       # 代码运行
├── debugger/     # 调试器
└── reviewer/     # 代码审查
```

功能：
- 代码编写
- 自动运行
- 代码审查

---

## 🔍 research-system

```
research-system/
├── search/       # 搜索
├── reader/       # 阅读
├── summarizer/   # 总结
└── analyzer/    # 分析
```

功能：
- 信息搜索
- 论文阅读
- 知识总结

---

## ⚙️ automation-system

```
automation-system/
├── tasks/        # 任务管理
├── workflows/    # 工作流
├── scheduler/    # 定时任务
└── monitor/     # 系统监控
```

功能：
- 自动化任务
- 工作流编排
- 系统监控

---

## 📋 productivity-system

```
productivity-system/
├── notes/        # 笔记
├── calendar/     # 日历
├── tasks/        # 待办
└── reports/      # 报告
```

功能：
