# 🔄 工作流程

## 示例：创建Python日志分析项目

```
用户输入: "创建一个Python日志分析项目"
```

### 执行流程

```
┌─────────────────────────────────────────────────────┐
│  👔 CEO                                              │
│  接收任务，分析需求                                    │
│  ↓ 决定调用 planner                                   │
└─────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  📋 PLANNER                                          │
│  拆解任务：                                          │
│  1. 调研日志分析需求                                  │
│  2. 设计项目结构                                     │
│  3. 编写核心代码                                     │
│  4. 代码审查                                         │
│  ↓ 调用 researcher                                   │
└─────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  🔍 RESEARCHER                                       │
│  搜索：Python日志分析最佳实践                          │
│  调研：常用日志库（loguru, logging）                   │
│  ↓ 返回调研结果                                       │
└─────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  💻 ENGINEER                                         │
│  使用 skill: project-generator                       │
│  创建项目结构                                         │
│  编写代码：                                           │
│  - main.py                                          │
│  - log_parser.py                                    │
│  - analyzer.py                                      │
│  ↓ 提交代码                                          │
└─────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  🔎 REVIEWER                                         │
│  使用 skill: code-reviewer                            │
│  审查代码质量                                        │
│  提出改进建议                                        │
│  ↓ 返回审查结果                                       │
└─────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────┐
│  📊 ANALYST                                          │
│  分析项目复杂度                                      │
│  估算学习时间                                        │
│  生成项目报告                                        │
│  ↓ 返回完成报告                                       │
└─────────────────────────────────────────────────────┘
                       ↓
                   ✅ 完成
```

## 调用Skills示例

| Agent | 调用的Skill |
|-------|-------------|
| CEO | - |
| Planner | task-manager |
| Researcher | web-search, knowledge-summarizer |
| Engineer | project-generator, python-runner |
| Reviewer | code-reviewer |
| Analyst | statistics-tool |

## 其他工作流

### 学习Python
```
CEO → Planner → Researcher → Engineer → Reviewer
```

### 解决Bug
```
CEO → Engineer → Reviewer
```

### 研究新课题
```
CEO → Researcher → Analyst
```
